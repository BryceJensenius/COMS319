CSS can be applied to HTML by the next three forms :

  • External stylesheet: The External Stylesheet is a separated file with extension .css and is linked to the actual HTML file.
  • Internal stylesheet: The CSS Internal Stylesheet is a section of code within an HTML document. It is not necessary write an external file .css and it is not required add a link into the section <head>...</head>.
  • Inline styles: The CSS Inline Style is used to apply a unique style to a single HTML element.

1) External Stylesheet

The External Stylesheet is a separated file with extension .css and is linked to the actual HTML file.

The content of the .css file make references to the HTML via tags, classes or ids then the appearance of the Web page is improve it.

The advantages to use an external stylesheet document per each HTML document are the next :

  • The same .css file can be linked to different HTML Web pages if necessary.
  • It is easy to maintain your html page.
  • Change and Look process is simplified.
  • HTML Web pages are shorter. Then .css file is cached on the browser side, so the Internet traffic is reduced.

We must include a link in our HTML code. The link to the .css External Stylesheet is as follows : 


                <link rel="stylesheet" href="namefile.css" >

Where namefile.css is the name of the External Stylesheet. If the file is in different path or folder from the actual HTML file we must include it in the file name.

2) Internal Stylesheet

The CSS Internal Stylesheet is a section of code within an HTML document. It is not necessary write an external file .css and it is not required add a link into the section <head>...</head>.

Let's write a HTML code including a CSS Internal Stylesheet.

The next code is a CSS Internal Stylesheet embedded in the <head>...</head> section.


                <!DOCTYPE html>
                <html>
                <head>
                   <style>

                      h2 {
                        color: whitesmoke;
                        background-color: brown;
                        font-family: cursive
                      }

                   </style>
                </head>
                <body>

                    <h3>1) External Stylesheet</h3>
                    <hr>

                    <h1>This is one example to show CSS External Stylesheet .</h1>
                    <p>This is p paragraph to improve appearance using CSS External Stylesheet CSS.</p>
                
                    <h1 class="myh1class">This is one example to show CSS External Stylesheet .</h1>
                    <p class="mypclass">This is p paragraph to improve appearance using CSS External Stylesheet CSS.</p>

                    <h3>2) Internal Stylesheet</h3>
                    <hr>

                    <h2>This is one example to show CSS Internal stylesheet. </h2>

                </body>
                </html>

Observe that the .css code is written directly inside the HTML document for improving the text of a tag h2.

The Internal Stylesheet has the next inconvenients :

  • If the .css code is extense, all the code will increase the size of the HTML document and its maintenance will become complex.
  • The .css is useful only in the actual document. If we need the same work in different HTML document, all .css has to be copied to the new HTML.

3) Inline Style

The CSS Inline Style is used to apply a unique style to a single HTML element.

The following example sets the text color of the <h1> element to blue, and the text color of the <p> element to red:


                <h3> 3) inline Stylesheet </h3>
                <h1 style="color:Green; background-color: transparent;">A Green Heading</h1>

                <p style="color:purple;">A purple paragraph.</p>

The style="color:Green;" will modify the appearance of the <h1> element in the HTML document.

And the style="color:purple;" will modify the appearance of the <p> element.

Summary :

CSS can be applied to HTML by the next three forms :

  • 1) External stylesheet
    by using a <link> element to link to an external CSS file.
  • 2) Internal stylesheet
    by using a <style>...</style> element in the <head>...</head> section.
  • 3) Inline styles
    by using the style attributes inside HTML elements.
