Activity 01 during Class - Create HTML using VSC

Open Visual Studio Code and create one initial HTML file.

Use the symbol ! to init one HTML code, similar to the next code :


    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
        <h1>Hello World !</h1>
    </body>
    </html>

Now, let's add the next tags :

  • Header h2 : <h2></h2>
  • Header h3 : <h3></h3>
  • Header h4 : <h4></h4>
  • Paragraph p : <p></p>
  • List ul : <ul></ul>
    • Li : <li></li>

    <!DOCTYPE html>
    <html lang="en">

        <h3>This is a Header h3 .</h3>

        <p>This is a default paragraph, compare the font type and font size.</p>
    
        <p>The Next is a List UL: </p>

        <ul>
            <li>Thing number 1.</li>
            <li>Thing number 2.</li>
            <li>Thing number 3.</li>
        </ul>
    </body>
    </html>
